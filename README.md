# 🏥 Patient Room Monitoring System
 
An IoT-based real-time monitoring system for healthcare facilities that tracks patient vital signs and room conditions in hospital environments.
 
---
 
## 📌 About Project
 
The Patient Room Monitoring System is an IoT application designed to monitor critical patient health metrics and environmental conditions in hospital rooms. This system collects real-time data from various sensors placed in patient rooms and displays them on a centralized dashboard. It helps healthcare professionals track patient vitals, room temperature, humidity, oxygen levels, and other critical parameters to ensure patient safety and comfort.
 
**Key Benefits:**
- Real-time patient health monitoring
- Automated alerts for critical conditions
- Remote patient tracking for multiple rooms
- Data-driven healthcare decisions
- Improved patient care quality
 
---
 
## 🛠️ Tools & Tech Used
 
| Component | Technology |
|-----------|-----------|
| **Microcontroller** | ESP8266 (NodeMCU) |
| **Sensors** | DHT11 (Temperature & Humidity), LDR (Light), Air Quality Sensor |
| **IoT Platform** | Blynk |
| **Communication** | WiFi |
| **Programming Language** | Arduino (C++) |
| **Libraries** | ESP8266WiFi, BlynkSimpleEsp8266, DHT, SimpleTimer |
 
---
 
## ✨ Key Features
 
- 🏥 **Real-Time Vital Monitoring** - Tracks patient heart rate, temperature, oxygen saturation, and blood pressure
- 📊 **Live Dashboard** - View data from multiple patient rooms on a centralized dashboard
- 🚨 **Automated Alerts** - Immediate notifications when vital signs exceed safe thresholds
- 🌡️ **Environmental Control** - Monitors room temperature, humidity, and air quality
- 📱 **Mobile Access** - Check patient status remotely via mobile app
- 💾 **Data Logging** - Historical records of patient vitals for medical analysis
- 🔔 **Multi-Room Tracking** - Monitor multiple patient rooms simultaneously
- 🔐 **Secure Access** - User authentication and role-based access control
- 📈 **Analytics & Reports** - Generate health reports and trends
- ⚡ **Low Latency** - Fast sensor data transmission and processing
